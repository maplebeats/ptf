##ptf是什么
* Ptf全称 Python titus frameworks
* 总结自己这一年多的时间学到的东西，开发一套完整的框架
* 初期目标是对某互联网公司某部门的某CPP框架的一个Python实现

##ptf有哪些功能(初步构想)
* 自动容灾
* 立体监控
* 二进制通讯(protobuff)
* 高效发布
* 直观展示状态
* 自动伸缩
* 自动化测试工具
* 日志集群
* 统一配置
* ......
